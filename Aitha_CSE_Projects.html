Project Title: Scratch Project

Project Start: August 29, 2017

Project Due: September 8, 2017

Partner: Zachary Diodati


Scratch, a website created by the Massachusetts Institute of Technology, is a simple coding environment wih drag and drop elements that are easy to work with for people who are new to computer programming. The Scratch Project was the first partner-oriented task that we were given in CSE (Computer Science/Software Engineering). We were told to make a fully functioning game, given some basic criteria. We also made a Scratch account, created a project, and shared it between me and Zachary so that we both had access to it. Our game was called "A Battle Against Grey". One important requirement for the Scratch Project, and most other projects, was to create an Engineering Notebook. This notebook included daily documentation of our progress throughout the given timeframe. It also contains our original ideas, a flowchart created with draw.io, instructions to our game, and comments. The notebook that we edited and worked on is linked here. Project Title: Image Manipulation

Project Start: March 6, 2018

Project Due: March 21, 2018

Partner: Jason Tran


In this project, we were asked to manipulate an image based on a chosen client's requirements. We used the PIL Imaging Library to add multiple manipulations. Our code worked on an entire folder of images that was provided by the user, showing that our manipulations were not unique to specific types of images. We chose McDonald's as our client, so two out of our three manipulations were centered around them. Our first manipulation blended the McDonalds logo and the current image from the folder, and the user was able to have interaction with the code by specifying a ratio between the opacity of the logo and the given image. Our second manipulation shrunk the McDonalds logo to 25x25, and then it pasted the mini image over and over around the given image to create a border made out of mini-logos. Our last manipulation was not really related to McDonalds, so it just inverted all the colors of the image. Throughout this project, we were asked to keep a documentation of all our work, which is linked here. We coded our manipulations in an online cloud IDE called Cloud 9, and the link to our workspace can be found here. Please feel free to use our manipulations in any way you want! You can change the logo that we used as well.
