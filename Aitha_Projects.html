Project Title: Scratch Project

Project Start: August 29, 2017

Project Due: September 8, 2017

Partner: Zachary Diodati


Scratch, a website created by the Massachusetts Institute of Technology, is a simple coding environment wih drag and drop elements that are easy to work with for people who are new to computer programming. The Scratch Project was the first partner-oriented task that we were given in CSE (Computer Science/Software Engineering). We were told to make a fully functioning game, given some basic criteria. We also made a Scratch account, created a project, and shared it between me and Zachary so that we both had access to it. Our game was called "A Battle Against Grey". One important requirement for the Scratch Project, and most other projects, was to create an Engineering Notebook. This notebook included daily documentation of our progress throughout the given timeframe. It also contains our original ideas, a flowchart created with draw.io, instructions to our game, and comments. The notebook that we edited and worked on is linked here. For this project, we had multiple drafts for our game. The first draft is linked here, while the second and final draft is linked here. Please feel free to comment, like, remix, and play our game in Scratch!
